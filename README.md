# 📐 Método de Eliminação de Gauss --- Calculadora de Sistemas Lineares

Este projeto é uma aplicação web que resolve **Sistemas Lineares**
utilizando o **Método de Eliminação de Gauss**, mostrando **todos os
passos do cálculo**.

O sistema permite:

-   Inserir qualquer matriz
-   Ver o escalonamento passo a passo
-   Classificar o sistema (SPD, SPI ou SI)
-   Calcular o valor das variáveis
-   Mostrar solução paramétrica quando houver infinitas soluções

------------------------------------------------------------------------

# 🚀 Como usar

1.  Escolha o número de **Equações**
2.  Escolha o número de **Variáveis**
3.  Clique em **Gerar Matriz**
4.  Preencha os valores
5.  Clique em **Resolver**

------------------------------------------------------------------------

# 🧠 Classificação dos Sistemas

## ✅ SPD --- Sistema Possível e Determinado

Possui **uma única solução**

## ♾️ SPI --- Sistema Possível e Indeterminado

Possui **infinitas soluções**

## ❌ SI --- Sistema Impossível

Não possui solução

------------------------------------------------------------------------

# 📊 O que o sistema mostra

✔ Troca de linhas\
✔ Cálculo do pivô\
✔ Zeramento abaixo do pivô\
✔ Matriz após cada passo\
✔ Substituição para trás\
✔ Classificação automática\
✔ Solução final\
✔ Solução paramétrica (SPI)
